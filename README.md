# Rub Web Libraries

A [Rub](https://github.com/kevincox/rub) library for common web build operations
such as minifying and combining source files.

## Installing

Use Rub of course:
```bash
cd rub-web
rub :install
```

## Docs

Once installed regenerate the Rub docs and rub-web will be included in your
documentation.
```bash
rub --doc
```
